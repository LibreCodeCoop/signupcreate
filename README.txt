Signup Create Course Moodle Plugin
==============================

Features
--------

This plugin lets you automatically create a course when a user sign up to Moodle.

Installation / Configuration
----------------------------

1. Add the plugin into /course folder.
2. Enable the enrolment plugin and set it up. It is quite similar to other enrolment plugins.

Use
----
Enable the plugin in the courses you want to auto-enrol users on sign up.
Once a new user is added to Moodle, he will be automatically enroled in the courses that have the signup plugin enabled.
